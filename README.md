# anxiety-test
Тест на тревожность
Здесь представлен Тест на тревожность, который состоит из 33 распространенных симптомов тревоги.
Инструкция
Отметьте, насколько каждый из перечисленных симптомов беспокоил вас в течение последней недели, используя следующую шкалу:
0 — ни разу;
1 — иногда;
2 — умеренно;
3 — часто.
Рекомендации
Еженедельный мониторинг: рекомендуется заполнять этот опросник еженедельно, чтобы отслеживать прогресс в ходе лечения или самопомощи.
Цель: Считается, что вы чувствуете себя превосходно, когда показатель по тесту не превышает 4 баллов. Любое снижение баллов является признаком улучшения состояния.
Связь с депрессией: Тревожность часто идет рука об руку с депрессией. Если ваши баллы по этому тесту высоки, рекомендуется также пройти Тест на депрессию
